# 🌸 Adrija P — Personal Portfolio

> *"Good skin, like good code, is built one layer at a time."*

A **premium, feminine-tech portfolio** built for Adrija — a CS student, Microsoft & Google enthusiast, skincare geek, and aspiring software engineer. Featuring a rose-gold, midnight-navy & lavender design system with interactive 3D WebGL visuals.

---

## ✨ Features

- **🎨 Feminine Design System** — Rose-gold, midnight-navy & lavender color palette with glassmorphism cards and smooth micro-animations
- **✦ 3D Particle Lab** — Interactive WebGL particle universe powered by Three.js (with webcam support)
- **⌨️ Typewriter Hero** — Animated role text cycling through her skills and passions
- **🗂️ Multi-Page Architecture** — Fully modular pages (About, Skills, Projects, Journey, 3D Lab, Contact)
- **🌀 Scroll Reveal Animations** — Elements fade in elegantly as the user scrolls
- **📱 Responsive Layout** — Looks great on all screen sizes
- **🚀 Vite-Powered** — Lightning-fast dev server and optimised production builds

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) v18+

### Installation

```bash
# Clone the repository
git clone https://github.com/Pushkar555E/adrija-.git
cd adrija-

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open your browser at `http://localhost:5173` to see the portfolio.

### Build for Production

```bash
npm run build
```
